Reserva de habitación

Esta opción le ofrece la reserva de una habitación solo para usted y quien lo acompañe, sin necesidad de compartirla con otros.

Para reservar una habitación, seleccione una fecha y pulse en reserva para dirigirte a la pagina de pagos.

Total: 0 €

Reserva de cama

Esta opción le ofrece la reserva de una o varias camas. Esto conlleva a compartir la habitación con otras personas.

Para reservar una cama, seleccione una fecha y la cantidad de camas que desea, tras ello, pulse en reserva para dirigirte a la página de pagos.

Total: 0 €

Tras realizar la reserva se procederá con el cobro de dicha 3 días antes del día de hospedaje. Una vez realizado el cobro no será posible solicitar el reembolso.
